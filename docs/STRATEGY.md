# Strategy

## Project Type

**SEO / Sales Website** (Type A)

This is an external-facing website built to generate traffic, leads, and bookings for private tutoring in Thailand.

## Project Purpose

Private Tutoring Thailand connects families with qualified private tutors across Thailand. The service is built for international families, expat parents, and local families using international curricula who need reliable, professional tutoring support.

## Mission

Create a fully automated SEO, content, landing page, and lead-generation system for private tutoring in Thailand. Dominate search coverage across country, city, subject, curriculum, and parent problem keywords.

## Content Strategy

### Page Types and Required Sections

Every page must include:
1. Clear headline (H1) with primary keyword
2. Parent problem statement
3. Service explanation
4. Who the service is for
5. Subjects or curriculum covered
6. Location relevance
7. Tutor matching explanation
8. Trust section
9. FAQ section (5-7 questions)
10. Clear call to action
11. Internal links (minimum 3)
12. Images following `docs/image-rules.md`

### Content Quality Standards

- Service pages: 800-1500 words
- Blog articles: 1200-2500 words
- No fake claims, testimonials, or credentials
- No duplicate city-swap pages
- Every page answers a real parent question

## Conversion Strategy

### Primary Conversion Actions
1. WhatsApp message
2. Contact form
3. Book a consultation
4. Request tutor matching

### CTA Placement
- Above the fold (first 300 words)
- After problem statement
- Mid-page
- Bottom of page

## Internal Linking Architecture

### Hub and Spoke Model
- **National Hub:** Home page → all locations, subjects, curricula
- **Location Hubs:** Location pages → subjects, curricula, blog articles
- **Subject Hubs:** Subject pages → locations, curricula, blog articles
- **Curriculum Hubs:** Curriculum pages → locations, subjects, blog articles

### Link Rules
- Every service page: minimum 3 internal links
- Every blog article: minimum 1 service page link
- Descriptive anchor text (no "click here")
- No orphaned pages

## Tracking Strategy

### Metrics
- Page visits by page type
- WhatsApp clicks by page
- Form submissions by page
- Conversion rate by page type
- Organic search traffic and rankings
- Lead quality by source

### Business Questions
1. Which cities create leads?
2. Which subjects create leads?
3. Which curricula create leads?
4. Which blog articles support conversions?
5. Which pages need improvement?

## Brand Voice

- Clear, professional, parent-friendly
- Trustworthy, direct, calm
- International, helpful
- Not childish, not corporate, not overpromising

## Competitive Position

**Gap:** Professional, curriculum-specific, Thailand-focused private tutoring with vetting and matching.

**Competitors:**
- Large tutoring centers (Kumon) — local curriculum, not international
- Marketplace platforms (Superprof) — no vetting, inconsistent quality
- School-recommended tutors — limited availability
- Online platforms (Preply) — not Thailand-specific

## Long-Term Goals

1. Dominate organic search for private tutoring in Thailand
2. Cover all 13 primary locations
3. Cover all core subjects and curricula
4. Build a blog system that supports SEO and conversion
5. Establish brand as the trusted choice for expat families

## Strategy Review Schedule

- **Weekly:** Traffic, leads, conversion rates
- **Monthly:** Keyword rankings, content performance, strategy alignment
- **Quarterly:** Full strategy review, market changes, competitor analysis
