# Backlog

## Purpose

Every idea, suggestion, or future task must be captured here.

When an idea is suggested:
1. Add it to this file
2. Rate it: Must have / Should have / Could have / Won't have
3. Estimate effort: Small / Medium / Large
4. Link it to the relevant area
5. Schedule it or leave it in the backlog

When an idea is scheduled:
1. Move it to `docs/6-MONTH-PLAN.md`
2. Create a task in `docs/TRACKING.md`
3. Mark it as scheduled in this file
