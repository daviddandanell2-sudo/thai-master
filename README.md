# private-tutoring-thailand
AI-controlled SEO, content, landing page, and lead-generation system for private tutoring in Thailand.

## Project Purpose

This repository is the central control system for building, managing, and scaling Private Tutoring Thailand.

The goal is to create a fully automated SEO, content, landing page, and lead-generation system for private tutoring in Thailand.

This is not just a website.

This is a market acquisition system.

The system must help us understand the Thailand tutoring market, create high-quality location pages, subject pages, curriculum pages, blog content, internal linking, tracking, and conversion flows.

The system must be built so AI agents, developers, SEO tools, and content workflows can all work from the same structure.

## Core Business Concept

Private Tutoring Thailand helps families, expat parents, international students, homeschool families, and relocating families find private tutors in Thailand.

The service focuses on:

* One-to-one private tutoring
* International curriculum support
* Home and villa tutoring
* Online tutoring
* Exam preparation
* SEN and learning support
* Homeschool support
* Short-term tutoring for families visiting Thailand
* Long-term tutoring for families living in Thailand

The business should feel trusted, professional, discreet, and easy for parents to understand.

The parent should quickly understand:

1. What we do
2. Where we do it
3. Which subjects we cover
4. Which curricula we support
5. How the tutor matching works
6. How to contact us
7. Why they can trust us

## Automation Goal

The system must allow AI agents to:

1. Research the Thailand tutoring market
2. Find search demand
3. Identify parent personas
4. Build keyword maps
5. Create location pages
6. Create subject pages
7. Create curriculum pages
8. Create blog plans
9. Generate internal links
10. Generate meta titles and descriptions
11. Generate FAQ sections
12. Generate image prompts
13. Check SEO quality
14. Check brand voice
15. Check page duplication
16. Update sitemap logic
17. Prepare pages for deployment
18. Track performance and improve the system

The AI must not randomly create content.

The AI must follow the file structure, templates, content rules, SEO rules, and quality-control system.

## AI Agent Rules

Every AI agent must read this README first.

Every AI agent must understand:

* This is a Thailand private tutoring system
* The structure must stay clean
* Every page must have a clear search purpose
* Every page must connect to related pages
* Every page must have conversion intent
* Every page must be useful for parents
* Every claim must be checked before publishing
* No fake statistics
* No fake testimonials
* No fake tutor credentials
* No duplicated city pages with only city names changed
* No thin SEO pages
* No random blog articles without a keyword or conversion purpose

## Content Quality Rules

Every page must answer real parent questions.

7. Tutor matching explanation
