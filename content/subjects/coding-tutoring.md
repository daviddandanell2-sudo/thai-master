## Frequently Asked Questions

**What age should my child start coding tutoring?**
Most coding tutoring begins around age 10, though younger students can start with Scratch. Older students often need support for formal computer science courses.

**Which curriculum does your coding tutoring follow?**
We adapt to your child's needs, whether that's IB Computer Science, IGCSE, American curriculum, or personal interest.

**Do you offer online coding tutoring?**
Yes, online coding tutoring works very well with screen sharing for writing and reviewing code together.

**How do I know if my child needs a coding tutor?**
Signs include struggling with computer science coursework, wanting to learn to code but not knowing where to start, or having project ideas but lacking the skills to execute them.

**Can a tutor help with IB Computer Science Internal Assessment?**
Yes, tutors can guide students through the IB Computer Science IA, helping with planning, coding, and documentation while ensuring the work remains the student's own.

**How long are coding tutoring sessions?**
Sessions are typically one hour. Project-based sessions may sometimes run longer.

**What should my child bring to a coding tutoring session?**
A laptop, any school computer science materials, project ideas or current code they're working on, and a notebook for planning.
